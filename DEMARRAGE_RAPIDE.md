# 🚀 Démarrage Rapide - Polarity Quiz

## ⏱️ 10 minutes pour déployer

### Étape 1 : GoatCounter (2 min)

1. Aller sur https://www.goatcounter.com/signup
2. Créer un compte avec le code `polarityquiz` (ou autre)
3. Noter votre code

### Étape 2 : Configurer le code (30 sec)

**Fichier** : `index.html` (ligne 11)

Remplacer :
```html
<script data-goatcounter="https://YOUR-CODE.goatcounter.com/count"
```

Par (exemple avec `polarityquiz`) :
```html
<script data-goatcounter="https://polarityquiz.goatcounter.com/count"
```

### Étape 3 : Tester en local (1 min)

```bash
cd "C:\Users\cherc\Downloads\my-politest\my-politest"
npm run build
npm run preview
```

Ouvrir http://localhost:4173 et tester

### Étape 4 : Déployer sur Vercel (5 min)

**Option simple** :
1. Aller sur https://vercel.com/new
2. Se connecter avec GitHub
3. Importer le projet
4. Cliquer sur "Deploy"
5. **Terminé !** 🎉

Votre site sera sur : `https://votre-projet.vercel.app`

### Étape 5 : Vérifier que ça marche (1 min)

1. Aller sur votre site déployé
2. Cliquer sur "Commencer le test"
3. Attendre 30 secondes
4. Aller sur `https://polarityquiz.goatcounter.com` (votre dashboard)
5. Vous devriez voir la visite ! ✅
